# Sarcasm-Detection
This project focuses on detecting sarcasm in text using Natural Language Processing (NLP) and Machine Learning/Deep Learning techniques. Sarcasm detection is a challenging task in sentiment analysis since sarcastic statements often express the opposite of their literal meaning.
